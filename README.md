# Multilingual Yes/No Answer Classification 

## Overview 
This project reimplements the paper: **“Interpreting Indirect Answers to Yes-No Questions in Multiple Languages”** 

The goal is to classify indirect answers to yes/no questions into: - Yes - No - Middle / Maybe 

## What I worked on 
- Built data preparation pipelines for English and multilingual datasets 
- Fine-tuned XLM-RoBERTa for sequence classification 
- Implemented training and evaluation loops in PyTorch 
- Reproduced experimental results from the paper across multiple languages 
- Analyzed model performance and error cases 

## Tech Stack 
- Python 
- PyTorch 
- Hugging Face Transformers 
- XLM-RoBERTa 
- Pandas / NumPy / scikit-learn 

## Results 
The project reproduced results across multiple languages including Hindi, Korean, Chinese, Turkish, Spanish, Nepali, and Persian, with several results matching or improving on expected benchmarks.
